# HOW TO RUN

You have to have the following hierarchy:

    registration.exe
    f/
    |
    `-+-- fixedImage1.dcm
      |
      +-- fixedImage2.dcm
      | 
      +- ...
      |
      +-- fixedImageN.dcm
    m/
    |
    `-+-- movingImage1.dcm
      |
      +-- movingImage2.dcm
      | 
      +- ...
      |
      +-- movingImageN.dcm
